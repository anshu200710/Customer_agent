## 1. Slow Digit Input (Chassis Collection)

**Conversation:**
Agent: Namaste ji, machine mein kya problem hai?  
Customer: Start nahi ho rahi ji.  
Agent: Chassis number bataiye ji.  
Customer: Teen... teen... char...  
Agent: Ji, 3 3 4. Aage bataiye.  
Customer: Paanch... ruko... bhai se pooch raha hun...  
Agent: Ji zarur, main hun.  
Customer: Haan... paanch sat do.
